RFC search provider

This application provides a very simple rfc search provider.

Usage

  1. First install the search provider, you can do this here:
  2. Then (in firefox) go to 'Manage Search Engines', and add the 'rfc' keyword for this search provider.
  3. Now you can easily search for rfcs by typing 'rfc [keyword]' in your addressbar. Examples:
    1. rfc http
    2. rfc 2616
    3. rfc webdav
